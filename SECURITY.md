## Reporting a Vulnerability

In order to report a vulnerability, you want to :
 - Indicate which version of the engine do you use
 - Which version of the librairies are you using
 - Which OS are you using
 - Which file is concerned if you've seen a vulnerability
 - AND/OR the scenario I would have to reproduce if I wanted to exploit the vulnerability
